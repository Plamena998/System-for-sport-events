System for sport events:

A comprehensive web-based platform for managing and organizing sports events, built with Laravel 11. The system provides role-based access control, allowing administrators to manage events, sport types, and organizers, while regular users can browse and search through upcoming sports events.

=> This system offers a central web platform where:
 - Administrators can create, manage, and monitor all sports events in one place;
 - Information about event organizers is stored in an organized database;
 - Sports are grouped by type and are easy to manage;
 - Users can view, search, and filter events by date, organizer, or sport type;
 - Images from events help make them more attractive and engaging;

=> Target Audience:
 - Sports clubs and sports organizations;
 - Companies that organize events;
 - Universities and schools that run sports programs;
 - Local community sports centers;
 - City and municipal sports departments;

=> Key Features

===================
For Administrators:

Event Management
Create new sports events with basic details such as name, date, duration, and description
Upload photos to make events more appealing
Edit event information when changes are needed
Delete events that are cancelled or finished
Search and filter events by name, date, organizer, or sport type
Sort events by name, date, or duration

Sport Types Management
Create and organize different sport types (Football, Basketball, Tennis, etc.)
Add short descriptions for each sport type
See how many events belong to each sport type
Prevent deleting sport types that are still used in active events

Organizers Management
Keep a list of all event organizers
Store organizer contact details such as email, phone, and address
See which events are managed by each organizer
Protect data by not allowing deletion of organizers with active events

User Management
Create user accounts
Set roles for users (administrator or regular user)
Control user access and permissions
Delete users when needed

=================
For Regular Users

Event Discovery
Browse all upcoming sports events
View event details such as date, time, duration, and location
See event photos and descriptions
Filter events by date
Filter events by organizer
Filter events by sport type
Use multiple filters at the same time for better search results
